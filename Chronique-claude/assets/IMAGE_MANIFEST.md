# Image manifest — La Chronique

All photography is shot by **Fany Ducharme** (credited in the site footer).

The host server `lachronique.qc.ca` does not return CORS headers, so these images
cannot be fetched and cached into this project automatically. They are referenced
**by URL** from the HTML in `preview/`, `slides/`, and `ui_kits/website/`.

For an offline-portable design system, please drop local copies into this folder.
